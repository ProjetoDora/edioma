# Edioma 
O Edioma é uma linguagem de programação em português criada para o Projeto Dora

## Fazendo download

Clone a aplicação
```
  git clone git@github.com:ProjetoDora/edioma.git
  cd edicodar/
```

## Iniciando a aplicação

Instale as dependencias npm e inicie a aplicação
```sh
  npm install
  npm start
```
## Rodando os testes 

```sh
  npm test
```
